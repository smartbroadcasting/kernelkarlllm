# MCP Roadmap

MCP is not implemented in Phase 1. The repository already reserves the structure for future adapters:

```text
mcp/
├── sql/
├── drupal/
├── foosbook/
├── osticket/
├── filesystem/
└── news/
```

Planned servers:

- SQL
- Drupal
- Foosbook
- osTicket
- filesystem
- news sources

The public REST API should remain stable while MCP capabilities are added behind the service boundary.
